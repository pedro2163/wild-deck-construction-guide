Welcome to this Wild Deck Construction Page

Hey guys! This is a deck construction guide with every possible deck in the Wild format. It is really hard to maintain alone, so if you guys want to contribute it will be very appreciated :D I can put your name in the section at the bottom.

How does it work?

I propose some possible replacement cards based on this meta and why those cards can be useful. Some cards are flexible too, this means that they are not nescessary in the deck.

In red: don’t craft those cards, only add them if you have them in your collection.

In orange: I would say don’t craft those cards, but if you really need them go for it.

In green: must-to-have cards, I still propose a replacement but those cards really make a difference.

At the right of the Budget deck you will see some numbers at the same level of the cards to change. That number is the priority order, so #1 is the first change to make (and the first card to remove). If you have in your collection, for example, the #3 card to replace you need to add it! Don’t wait to have the #1 and #2 to add it after. If this happens, don’t remove #1 and add #3, you still need to remove #3 and add #3!

Normally if you can build the Semi Budget version you have a very strong deck, the final version it’s just a bonus.

The Budget decks don’t have Epics and Legendaries unless they are Really a need in the deck.

If you want to know which cards you can dust, just see this collection. I recommend you to keep every card in that collection.

Enjoy :)

How YOU can contribute?

If you spot some errors, or want to add more information (more possible cards or the Best order in the version story for example) just send me a message or ask for a Pull Request ^^. I did not make everything. Because of the new set I needed to do every deck again... So if you guys think that this project will be helpful, I encourage you to contribute with your ideas! If you want to be part of this project just send me a message! I'm using GitHub to do this. Well, I hope this can be helpful and you guys will appreciate it!
